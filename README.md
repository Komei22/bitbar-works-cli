# bitbar-works-cli

bitbar-works-cli is a bitbar plugin to check in/out the office.

## Install
1. Install bitbar

```
$ brew cask install bitbar
```

2. Install bitbar-works-cli

```
$ go get -u github.com/Komei22/bitbar-works-cli
$ bitbar-works-cli install
```
